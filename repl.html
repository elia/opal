Opal REPL

This page provides a simple REPL that is built into the opal build tools and is easily loaded on any page with:

opal.browser_repl();

The REPL simply takes single lines of ruby and evaluates them against the top level object in ruby. There are some limitations to this repl at the moment, which includes the inability to have multi line statements (each statement is evaluated once return is pressed). Also only the core opal libraires are loaded, so there is no access to RQuery and other libraries.

Ok, load REPL!
